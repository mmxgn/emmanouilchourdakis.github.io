Misc

A tutorial for a Connect 4 agent

A kaggle notebook with a step-by-step approach and lots of comments on implementing a Negamax agent with alpha beta pruning, memoization and an efficient heuristic. I also include the full source code for my agent and instructions on how to use it. You can check the notebook and also see the agent's current standing in the leaderboard.

Crete 1910-1920 colorized

A colorization I did using the free release of #DeOldify. The video is about my home island of Crete during the 1910-1920s. You can find it in my YouTube channel.

Dosbox PKGBUILD with the ne2000 driver

A PKGBUILD for Arch Linux where I adapted an older patch for the ne2000 driver that allows dosbox to access a host network interface. Fun to connect your Windows 3.11 dosbox to the exciting new world of the Internet. Usable with Winsock Trumpet or Arachne.
